การสร้าง Theme

June 20, 2015

เชื่อว่าหลายๆ คนอาจจะไม่ชอบธีมเดิมๆ ที่ Yii 2 มีมาให้ ซึ่งเป็น Bootstrap ส่วนตัวผมเองมองว่ามันก็สามารถใช้งานได้ดี และมีคอมโพเน็นส์มาให้ครบ แทบจะไม่ต้องหาอะไรเพิ่ม แต่การใช้งานจริงๆ เราคงไม่สามารถใช้แค่ธีม Bootstrap แบบเดิมๆ ได้ (มันคงเบื่อกันบ้างแหละ ^ )

วันนี้จะแนะนำการเปลี่ยนธิมของ Yii 2 ซึ่งทำได้ง่ายมาก แค่เปลี่ยนคอนฟิกใน main.php ในส่วนของ components -> view ให้มันชี้ไปที่พาทที่เราเก็บธีมของเราไว้ (แค่นี้เองหรอ ดูเหมือนง่าย ง่ายเฉพาะคอนฟิกครับ ฮา..)

'view' => [
     'theme' => [
         'pathMap' => [
            '@app/views' => '@app/themes/adminlte'
         ],
     ],
],

หลัการทำงาน

หลักการทำงานของ views ใน Yii 2 เมื่อ Web Application ของเราทำงานมันจะไป render ไฟล์ view ที่โฟลเดอร์ดีฟอลด์ views ก่อน จากนั้นก็จะเช็คว่ามีการเรียกใช้งานที่คอนโทรเลอร์ใหน มันก็จะไปค้นที่โฟล์เดอร์ชื่อตามคอนโทรเลอร์นั้นในไฟลเดอร์ views

ในการสร้างธีม เราก็แค่ตั้งค่าให้มันเปลี่ยนเส้นทางของการเรียกใช้งาน view จากปกติที่มันไปค้นทีโฟลเดอร์ views หรือ @app/views นั้น เราก็เปลี่ยนให้มันชีไปที่โฟล์เดอร์ธีมของเรา แค่นั้นเอง

หากมีการเรียกใช้งาน view ใหน และไปค้นในธีมที่เราชีไปแล้วไม่พบ มันจะกลับมาเรียกใช้งานที่พาทดีฟอลด์เดิมคือ views หรือ @app/views นั้นเอง
